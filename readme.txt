== Installation ==

1. Unzip and Upload `llc-tax.zip` to the `/wp-content/plugins/` directory inside of a folder named "/llc-tax" 

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Then go to the Widgets section of your wordpress site and simply drag and drop the widget titled "LLC Tax" to any widget area of your website.
